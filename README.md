# 🧠 Code Explainer App using Watsonx AI

This is a **full-stack web application** that allows users to **sign up**, **log in**, and **paste code snippets** (such as Python or JavaScript) to receive **plain-English explanations** using **IBM Watsonx AI**.

🔗 **Live Project**: https://code-explainer-app.onrender.com

## 🚀 How to Run the Project Locally

### 1️⃣ Clone the Repository
git clone https://github.com/Ronakrathore111/code-explainer-app.git
cd code-explainer-app

2️⃣ Install Dependencies
npm install

3️⃣ Create a .env File (Root Folder)

Create a file named .env and add:

IBM_API_KEY=your_ibm_api_key
IBM_PROJECT_ID=your_project_id

4️⃣ Start the Server
node server.js


Open your browser and visit:

http://localhost:3000

📌 Notes

The application uses IBM Watsonx AI for code explanation.

Authentication is handled using a simple JSON-based user store.

This project is intended for learning and academic purposes.
